# GitHub 上传说明（中文）

这个项目包可直接上传到你的 GitHub 仓库。

## 推荐做法

1. 打开你的仓库主页
2. 点击 **Add file → Upload files**
3. 将本项目包中的所有文件和文件夹拖入上传区域
4. 等待上传完成后点击 **Commit changes**

## 当前已包含内容

- `README.md`：项目首页介绍
- `assets/teaser.jpeg`：首页展示图
- `assets/original_figures/`：从论文修订稿中提取的若干候选图片
- `assets/framework_candidate.jpeg`：候选框架图
- `assets/visual_comparison_candidate.jpeg`：候选可视化对比图
- `assets/tev_visualization_candidate.jpeg`：候选 TeV 图
